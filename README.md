# Catálogo de Videojuegos

## Descripción del proyecto
Este proyecto es una aplicación web desarrollada con el patrón **MVC (Model-View-Controller)** en **ASP.NET Core**.  
Su objetivo es gestionar un catálogo de videojuegos, permitiendo listar títulos, filtrarlos por género, ver detalles y agregar nuevos juegos mediante un formulario.  

La aplicación está pensada como práctica académica para comprender la arquitectura MVC y el flujo de datos entre modelos, controladores y vistas.
---
# Tecnologías utilizadas

- **ASP.NET Core MVC** (.NET 10.0)
- **C#** para la lógica de negocio y controladores
- **Razor Pages** para las vistas dinámicas
- **Bootstrap** para el diseño responsivo y estilizado
- **CSS personalizado** para la apariencia oscura con acentos en púrpura y amarillo
- **Visual Studio 2026** como entorno de desarrollo
